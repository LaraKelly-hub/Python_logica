📞 Sistema de Atendimento

Sistema desenvolvido em Python para simular atendimentos de clientes.

Funcionalidades

- Cadastro de atendimentos
- Cálculo de tempo médio
- Identificação de atendimentos acima da média
- Contagem por tipo de atendimento
- Identificação do atendimento mais demorado

Tecnologias

- Python
- Estruturas de dados (listas e dicionários)
