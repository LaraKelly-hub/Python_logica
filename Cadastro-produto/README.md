# Cadastro de Produtos em Python

Este projeto é um exercício de lógica de programação utilizando Python.

Funcionalidades:
- Cadastro de produtos
- Registro de preço
- Registro de quantidade em estoque
- Cálculo do valor total do estoque
- Identificação do produto mais caro

Estruturas utilizadas:
- listas
- dicionários
- classes
- tratamento de exceções
