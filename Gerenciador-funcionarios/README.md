# Gerenciador de Funcionários em Python

Projeto desenvolvido para praticar manipulação de listas, dicionários e Programação Orientada a Objetos em Python.

## Objetivo

Criar um sistema simples para registrar funcionários e armazenar seus dados.

## Conceitos aplicados

- Listas
- Dicionários
- Classes
- Organização de dados em memória
