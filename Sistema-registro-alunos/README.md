📚 Sistema de Registro de Alunos

📌 Descrição

Este projeto consiste em uma simulação de um sistema de registro de alunos desenvolvido em Python.

O sistema permite ao usuário inserir os dados dos alunos (nome e notas), realizando validações para garantir a integridade das informações e armazenando os dados de forma estruturada. Ao final, o programa exibe os alunos cadastrados, bem como sua situação (aprovado ou reprovado) com base na média das notas.

---

🎯 Funcionalidades

✔️ Cadastro de alunos

✔️ Validação de entrada de dados

✔️ Cálculo de média automaticamente

✔️ Listagem de alunos cadastrados

✔️ Separação entre alunos aprovados e reprovados

---
